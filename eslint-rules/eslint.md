## array-callback-return

- `object`
  - `"allowImplicit" : boolean`
    デフォルト: `false`
    暗黙の `return undefined` を許可する
  - `"checkForEach" : boolean`
    デフォルト: `false`
    `forEach` メソッドのコールバックで `return` を禁止する

| extends      | value   |
| ------------ | ------- |
| `eslint:all` | `error` |
